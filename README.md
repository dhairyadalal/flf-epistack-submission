# Epistack

Epistack is an evidence-review prototype for exploring how framing and
source-policy choices shape an assessment. A **case** is a corpus of evidence
collected to support an inquiry. The site lets you switch between framings or
evidence policies, inspect the underlying evidence records, and ask a critical
inquiry chat about the analysis.

Two cases are included:

- **COVID origins** — a review of evidence on whether COVID-19 emerged from a
  natural spillover or a research-related incident.
- **Eggs** — a demonstration of how "are eggs good to eat?" produces different
  evidence requirements depending on how "good" is framed.

**Live demo:** https://flf-epistack-submission.onrender.com/

## Build and run

Requires Python 3.11+.

```bash
python -m pip install -e '.[test]'
python -m epistemic_audit.build
python -m http.server --directory site 8000
```

Open `http://localhost:8000`.

### Enabling the inquiry chat

The chat panel calls an LLM with the full case context and needs a server to
keep the API key off the client. Get an NVIDIA API key, then run:

```bash
python -m pip install -e '.[chat,test]'
cp .env.example .env
# edit .env and set NVIDIA_API_KEY to your key
epistack-serve
```

Open `http://localhost:8000`. Never commit a real key or put it in `site/app.js`.
