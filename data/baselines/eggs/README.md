# Eggs starter baseline

This is a deliberately small analogue of the imported COVID baseline. It does
not try to answer the unqualified question “Are eggs good to eat?” Instead it
shows that several legitimate operational questions select different evidence
and produce differently shaped conclusions.

The four evidence notes are curated examples, not a systematic review. Their
purpose is to exercise the same ingestion and annotation machinery across
population health, individual response, animal welfare, and environmental
impact. Sources were selected from peer-reviewed or authoritative publications;
the resulting notes still require independent domain review.
